2. Shift all zeros in a list to the end while maintaining the order of other elements :
def move_zeroes(arr):
    pos = 0
    for i in range(len(arr)):
        if arr[i] != 0:
            arr[pos], arr[i] = arr[i], arr[pos]
            pos += 1
    return arr
print(move_zeroes([0, 1, 0, 3, 12]))  

OUTPUT :
